# 纯前端演示说明

## 🎯 什么是纯前端演示？

纯前端演示是本项目提供的最简单、最友好的体验方式。它是一个独立的HTML文件，**无需安装任何软件**，只需要一个现代浏览器即可运行。

## 🚀 优势特点

### ✅ 零依赖
- 无需安装Node.js
- 无需运行npm install
- 无需配置环境变量
- 无需启动服务器

### ✅ 即开即用
- 双击批处理文件即可启动
- 浏览器自动打开演示页面
- 无需等待编译或构建

### ✅ 完全离线
- 本地模拟模式完全离线工作
- 无需网络连接
- 所有数据本地生成

## 📋 使用方法

### 步骤1：下载项目
将整个项目文件夹下载到本地计算机。

### 步骤2：启动演示
双击 `纯前端演示.bat` 文件。

### 步骤3：体验功能
浏览器会自动打开演示页面，您可以：

#### 🤖 AI课程推荐
1. 点击"🚀 生成AI推荐"按钮
2. AI会分析并推荐适合的课程
3. 查看推荐理由和匹配度评分

#### 📅 AI学业规划
1. 点击"📋 生成学习规划"按钮
2. AI会制定个性化的学习计划
3. 查看学期安排和进度跟踪

#### 💬 AI对话助手
1. 在输入框输入学习问题
2. 或点击快捷问题标签
3. AI会实时回复并提供指导

## 🔧 工作模式说明

### 本地模拟模式（默认）
- 所有AI功能在本地上模拟
- 无需网络，无需API密钥
- 适合快速体验和演示

### API联网模式
- 可连接真实的AI服务
- 需要配置API地址
- 适合测试真实AI功能

## 🎨 界面特点

- 现代化UI设计
- 响应式布局，适配不同屏幕
- 流畅的动画效果
- 直观的操作体验

## 🛠️ 技术实现

- **纯HTML/CSS/JavaScript** - 无框架依赖
- **原生Fetch API** - 网络请求
- **本地状态管理** - 无需外部库
- **模块化代码结构** - 易于维护

## 📞 故障排除

### 浏览器无法自动打开
如果批处理文件无法自动打开浏览器：
1. 手动打开 `demo.html` 文件
2. 或在浏览器中拖拽 `demo.html` 文件

### 功能无法正常工作
1. 确保使用现代浏览器（Chrome、Firefox、Edge等）
2. 尝试刷新页面
3. 检查浏览器控制台是否有错误信息

### 网络连接问题
- 本地模拟模式不需要网络
- API模式需要确保API地址正确且可访问
