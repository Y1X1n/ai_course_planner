# 🚀 AI智能课程规划与学业辅助系统 - 使用指南

## ⚡ 快速使用

1. **启动演示**: 双击 `本地演示.bat`
2. **打开浏览器**: 访问 http://localhost:3000
3. **开始体验**: 点击"一键体验AI功能"
4. **探索功能**: 在标签页中体验不同AI功能

## 🎯 功能说明

### AI课程推荐
- 点击查看智能推荐的课程
- 每门课程都有评分和推荐理由
- 基于专业、兴趣自动匹配

### 学业规划
- 自动生成学习计划
- 包含多个学期的安排

## ❓ 常见问题

### Q: 启动失败怎么办？
**A**:
1. 检查Node.js是否正确安装
2. 确认端口3000和5000未被占用
3. 尝试运行清理脚本: `cleanup.bat`
4. 重新安装依赖: `npm run install:all`

### Q: 数据库连接失败？
**A**:
1. 安装MongoDB或使用Docker版本
2. 检查MongoDB服务是否运行
3. 确认连接字符串正确
4. 使用模拟数据模式（无需数据库）

### Q: AI功能无法使用？
**A**:
1. 检查网络连接
2. 确认OpenAI API密钥配置
3. 在演示模式下使用模拟AI回复
4. 查看浏览器控制台错误信息

### Q: 界面显示异常？
**A**:
1. 清除浏览器缓存
2. 使用Chrome浏览器
3. 检查屏幕分辨率
4. 尝试重新加载页面

### Q: 文件上传失败？
**A**:
1. 检查文件大小限制（最大10MB）
2. 确认文件格式支持
3. 验证网络连接稳定
4. 查看服务器日志错误
